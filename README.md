## 💻 Projeto

Esse é um projeto web, responsivo, de uma cápsula do tempo que exibe memórias em uma timeline.

## 🚀 Tecnologias

Esse projeto foi desenvolvido durante a NLW da Rocketseat, Maio de 2023, com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
